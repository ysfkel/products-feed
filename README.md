# Products Feed Application
Application for real time tracking of products feed

## To run the application:
*   clone the repository
*   from the directory root run 'bower install' to install client side dependencies 
*   from the directory root run 'npm install' to install npm dependencies 
*   run 'npm start' to start the application
*   open web browser and navigate to 'http://localhost:3000/#/product-list'

## Project dependencies
*   AngularJS,NodeJS, Express, Socket.IO, Webpack, Jasmin, Karma, Mocha,Twitter Bootstrap etc

##Features

* Retrieves Products Feed from specified URL and returns retrieved product feed via a websocket connection
* On the client side, product feed is stored on web brower local storage for future processing 
* Updates UI with products list 
* Updates menu with retrieved products categories
* View List of all products
* View Selected Product in a modal popup
* View Products by selected product category
